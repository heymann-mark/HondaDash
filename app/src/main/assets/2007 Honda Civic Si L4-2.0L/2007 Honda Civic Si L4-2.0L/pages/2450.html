Component Tests and General Diagnostics

Cruise Control Input Test

NOTE: Always make sure you have the latest HDS software before doing the input tests.

1. Connect the Honda Diagnostic System (HDS) to the data link connector (DLC).
2. Turn the ignition switch ON (II).
3. Make sure the HDS communicates with the vehicle and the engine control module (ECM). If it doesn't, troubleshoot the DLC circuit.
4. Go to PGM-FI, and check for DTCs.







5. Do the following tests while monitoring parameters in the PGM-FI DATA LIST with the HDS.
