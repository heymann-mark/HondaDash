Thermostat Housing: Service and Repair

Water Outlet Removal and Installation

Removal
1. Drain the engine coolant.
2. Remove the air cleaner assembly.
3. Remove the harness holder (A) from the bracket, then remove the harness holder bracket (B).





4. Remove the upper radiator hose (A), heater hose (B), and water bypass hose (C).





5. Remove the air cleaner housing brackets.





6. Remove the water outlet.





7. Remove the EGR plate.





Installation
1. Install the EGR plate (A) with a new gasket (B).





2. Install the water outlet (A) with a new gasket (B).





3. Install the air cleaner housing brackets.





4. Install the upper radiator hose (A), heater hose (B), and water bypass hose (C).





5. Install the harness holder bracket (A), then install the harness holder (B).





6. Install the air cleaner assembly.
7. Refill the radiator with engine coolant, and bleed the air from the cooling system with the heater valve open.
