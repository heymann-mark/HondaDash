Navigation cannot control XM radio

Navigation cannot control XM radio

- Review the audio system diagnostic section.
- If you can only tune to channels 000, 001, 174, and 247, make sure the audio unit is set to the channel mode (see owner's manual), if is set to channel mode, call XM satellite radio customer support and check the account activation status.

1. Substitute a known-good navigation unit, and recheck.

Can the navigation control the XM radio?

YES - Replace the navigation unit.

NO - Do the audio system troubleshooting.
